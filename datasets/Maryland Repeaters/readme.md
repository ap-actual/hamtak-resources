# Maryland Repeaters
A dataset of Maryland repeaters, original data is from repeaterbook generated on 2026.01.18. 

## How To Use
Load the .kmz file into ATAK, Google Earth, or any other favorite app that takes .kmz files. 

## How to Contribute
Simply edit the maryland_repeaters.csv file and then use tab #2 in the tools/kml_generator tool to create a new .kmz file.
